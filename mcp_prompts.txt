# MCP Tool Test Prompts

Use these prompts to test whether the expanded MCP tools are working across analyst estimates, ratings, valuation, DCF, calendar data, company overview, peers, ESG, historical returns, technical indicators, and full research bundles.

---

## 1. Analyst Estimates / Ratings

```text
What are analyst estimates for AAPL for the next two fiscal years? Include revenue, EPS, and growth expectations.
```

```text
Compare MSFT and NVDA using analyst estimates, price target consensus, ratings snapshot, and stock grades.
```

```text
For AMZN, summarize analyst expectations, price target upside, and the current ratings snapshot.
```

```text
Which has the better analyst setup right now: AAPL, MSFT, or GOOGL? Compare estimates, ratings, and price target upside.
```

## 10. Debugging Prompts

Use these when you want to confirm whether one tool family is working.

### Analyst Tool Debug

```text

```

```text
Pull the ratings snapshot for MSFT and explain what fields were returned.
```

```text
Pull price target consensus for NVDA and show consensus, high, low, median, and implied upside if available.
```

### Valuation Tool Debug

```text
Pull DCF and levered DCF for AAPL and show the values returned before giving a conclusion.
```

```text
Pull valuation data for MSFT and tell me whether DCF, price target, and trading multiples were available.
```

### Calendar Tool Debug

```text
Pull recent earnings calendar data for AAPL and show the returned event dates.
```

```text
Pull dividend history for MSFT and summarize the latest five dividend records.
```

### Company / Peer Tool Debug

```text
Pull company profile, market cap, and peers for NVDA and show what fields were available.
```

```text
Find MSFT peers and compare only the data fields that are available from the MCP tools.
```

### Technical Tool Debug

```text
Pull technical indicators for NVDA and show which indicators returned data versus which were unavailable.
```

```text
Pull RSI and moving averages for AAPL and explain the current technical signal.
```

### Historical Price Tool Debug

```text
Pull 5-year historical price data for AAPL and show the start price, end price, and calculated return.
```

```text
Compare AAPL and MSFT 5-year price returns and show the calculation inputs.
```
